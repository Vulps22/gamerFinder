  • Online Safety

    • The Report Abuse button can be found on every public profile page.
    • You can use it when you believe someone has broken the rules that we all agreed to in the Privacy Policy & Terms of Use
    • We have also fixed an annoying issue with the registration process. so new users should now be able to access their profiles without difficulties

  • MessengerV1

  • Good news everyone! The messenger feature is now functional!

    Features:

    • You can now send private messeges! - When you've found someone to game with you, You're going to need a way to give them information (VOIP, game server etc)
    • To use the messenger you need to click the "contact" button on the desired user's public profile
    • When you do that the MessengerV1 will send you to the Messeges page. If you have never contacted this person before, a new thread will be created.
    • This is just the first implementation of Messeneger. MessengerV2 will be more intuitive, with features like automatically selecting the desired thread, and telling the user which request you accepted.

  • Delete Requests

  • In the latest update to Gamer Finder We've worked a little towards expanding the requests feature. This is just a small update

    Features:

    • You can now delete Requests - When you've found someone to game with you, you may want a way to stop looking for more gamers. This is how you do it.
    • It's simple, At any time you can open your profile where you will now see 2 new buttons. "Games" and "Requests"
    • click on Requests and you will be able to choose which request you want to delete.

  • Closed Beta

  • This is the first release of Gamer Finder.

    Features:

    • Sign in through steam. - We've managed to implement a "one-click" sign in with steam
    • View a list of your games
    • View a list of all games Gamer Finder knows about
    • Every time you browse to your profile page, Gamer Finder looks for new games on your steam list and adds them to it's database
    • You can submit an "I want to play" request for any game, Even if you don't own that game (Future releases of Gamer Finder may implement restrictions to prevent this)
    • You can view another player's public profile. Which includes a list of all the requests he/she has sent.

    As this is a "closed beta" please do not pass the address out to anyone. Thanks
