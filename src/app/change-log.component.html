  • Closed Beta

  • This is the first release of Gamer Finder.

    Features:

    • Sign in through steam. - We've managed to implement a "one-click" sign in with steam
    • View a list of your games
    • View a list of all games Gamer Finder knows about
    • Every time you browse to your profile page, Gamer Finder looks for new games on your steam list and adds them to it's database
    • You can submit an "I want to play" request for any game, Even if you don't own that game (Future releases of Gamer Finder may implement restrictions to prevent this)
    • You can view another player's public profile. Which includes a list of all the requests he/she has sent.

    As this is a "closed beta" please do not pass the address out to anyone. Thanks


    Bugs and Incomplete features:

    • Private Messages(Incomplete Feature)
    • Online Safety(Incomplete feature)
      • The "Report Abuse" button on public profiles does not work yet.
      • Until the online safety features for under 18's are complete, Under 18's will not be allowed to join Gamer Finder
    • The navigation bar does not expand from it's collapsed state when activated on mobile devices.
    • Sometimes stuff just doesn't function and there's no rhyme or reason as to why

    Planned changes for next release:

    • Give users the ability to delete their requests
    • Give users the ability to "Accept" another user's request - This should start a message thread with the user and email the Request sender
    • Give user private profile a "requests" tab to view a list of all his/her requests
    • Ability to create a new message thread with users
